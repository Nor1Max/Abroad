Для head:

<meta name="author" content="Максим"> - автор страницы  
<meta name="decription" content="Производится тест"> - описание страницы
<meta name="keywords" content="Тест, проверка, html"> - ключивые слова
(Все эти meta выше используются для поисковой системы)

<meta charset="UTF-8"> - кодировка страницы на Английскую и Русскую раскладку
<meta name="viewport" content="width=device-width, initial-scale=1.0"> - расширение страницы
<link rel="stylesheet" href=""> - для подключение CSS файлов


Для <body></body>:

<script src=""></script> - для подключения JavaScript кода (прописывайте в самом конце перед </>body>)
    

Для текста:

<p></p> - написать текст c новой строки
<div></div> - блочный элемент для будущего использования CSS
<span></span> - строчный элемент для будущего использования CSS
<b></b>, <strong></strong> - текст жирным
<i></i>, <em></em> - текст курсивом
<strike></strike> - перечёркнутый текст
<sub></sub> - сместить текст ниже
<sup></sup> - сместить текст выше
<br> - перенести новый текст на следующую строчку
<big></big> - делает текст большим
<small></small> - делает текст маленьким
    
    title="" - будет появляться подсказка для текста при наведении на него (можно поставить куда угодно)
    <pre></pre> - выводит текст так, как он был написан (отступы, пробелы, пропуски строк)

<code></code> - для написания строчки кода
<abbr></abbr> - для аббревиатуры
<address></address> - для адреса


Списки <li><li>:

<ol></ol> - пронумерованный список
    start="" - с какой цифры начнётся список 
    type="" - изменить отображение цифр (1 - цифры, a - английский, i - римиский)

<ul></ul> - непронумированный список
    type="" - изменить отображение кружочков (circle - выколотые, square - квадраты)
    

Ссылки:

<a href=""></a> - для ссылок 
    Для прехода на сайт вставьте обычку ссылку
    Для перехода на другой html файл пишите "название файла"
    Для перехода на самой странице пишите #"название"(сначало укажите <a name=""></a>)
    Для перехода по почте пишите mailto:"почтовый адрес"

target="" - переход на ссылку (_self - c этой же вкладки (по умолчанию), _blank - на новую вкладку)


Изображения, аудио и видеофайлы:

<img src=""> - для картинок URL (для использования фото с вашего ПК введите его расположение)
    width="" - для формата изображений (в % или в px)
    height="" - изменение только по высоте 
    alt="" - отображение надписи, если фото не загрузилось
<audio src="" controls></audio> - для аудио
<video src="" controls></video> - для видео
    controls - для возможности упровления видео/аудио


Таблицы:

<table></table> - для таблиц
    border="" - размер обводки таблицы (в px)
    width="" - ширина таблциы(в px)
    align="" - расположение (left, center, right)
<thead></thead> - описание шапки таблицы
<tbody></tbody> - описание основной части таблицы
<tfoot></tfoot> - описание футера таблицы
    <tr></tr> - описание ряда таблицы
    <th></th> - заголовок столбца
    <td></td> - обычный текст для столбцов


Формы и поля:

<form action="" method=""></form> - для форм 
    action="" - будет обрабатывть данные поступающие из формы
    method="" - метод предечати информации (get, post)
<label></label> - создание подписи для поля
    for="" - для отоброжения поля, для которого будет работать этот текст (работает только с id="")
<input type=""> - создание поля ввода информации (type="" - тип ввода)
       
        <input type="text"> - для обычного текста
        <input type="password"> - для пароля (невидимый текст)
        <input type="email"> - для почты
        <input type="url"> - для URL адреса
        <input type="color"> - для выбора цвета (не поддерживается на некоторых браузерах)
        <input type="date"> - для выбора даты (не поддерживается на некоторых браузерах)
        <input type="time"> - только для времени (не поддерживается на некоторых браузерах)
        <input type="datetime"> - для даты и времени (не поддерживается на некоторых браузерах)
        <input type="file"> - для файлов
        <input type="checkbox"> - для галочки
        <input type="radio"> - для кружочка
            name="" - введите одинаковое имя для обоих кружков, чтобы при нажатии на них горел только один
        
        <input type="button" value=""> - просто кнопка
        <input type="submit" value=""> - кнопка, с перезагрузкой сайта
        <input type="submit" value=""> - кнопка, очищающая полностью весь выбранный или написанный вами текст на сайте
        
    placeholder="" - подсказка при вводе
    value="" - заранее установленный текст при вводе
    maxlength="" - ограничение в символах
    hidden - спрятанное поле 
    disabled - неактивное поле (в которое нельзя ничего ввести)
     readonly - активное поле (в которое нельзя ничего ввести)
    name="" - для нахождения атрибутов в JavaScript
    id="" - для использования CSS или для обращения к данному элементу

<textarea></textarea> - для создания большого текстового поля
<button></button> - более удобный способ создания кнопки


Селекторы выбора:

<select></select> - список выбора
    multiple - множественный список (можно выбрать несколько элементов зажатым ctrl)
    size="" - количество вариантов показано пользователю 
<option></option> - элемент внутри списка
    selected - заранее выбранный элемент внутри списка
    disabled - выключенный элемент внутри списка


Специальные HTML5 теги:

<h1></h1> - самый важный заголовок
<h2></h2> - заголовки
<h3></h3> - внутри section
<header></header> - для шапки сайта
	<nav></nav> - навигация на сайте
<main></main> - для основной части сайта
    <section> - для секций на сайте
    <aside></aside> - для боковой части сайта
<footer></footer> - для футера

<progress></progress> - прогресс бар
    max="" - максимальная величина бара
    value="" - текущая величина бара
<details></details> - скрытый текст, который показывается после нажатия на определённую кнопку
    <summary></summary> - название определённой кнопки

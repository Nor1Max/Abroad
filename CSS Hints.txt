Обращение к HTML объектам:

"Селектор" {"Стиль;"} - правильное наложение стилей к html
#"название в id" {"Стиль;"} - наложение стилей к id
."название в class" {"Стиль;"} - наложение стилей к class

"Обращение к одному элементу" "Внутри этого элемента" - div b
"Обращение к одному элементу", "обращение к другому элементу {"Стиль"} - div i, div b, #id
"Обращение к одному элемент".text - обращение к элементу, в котором есть текст



Стили:

color: ; - цвет
height: ; - высота
width: ; - ширина
border: ; - обводка (1 - ширина, 2 - стиль обводки, 3 - цвет) (border-left - обводка слева и тд)
    solid - сплошная обводка
    dashed - прерывистая обводка
    dotted - обводка крожочками
border-radius: ; - скругление углов
border-style: ; - стиль обводки (double - двойная обводка)
opacity: ; - прозрачность (числа от 0 до 1 включительно)
margin: ; - внешние отступы со всех сторон (margin-top - отступ сверху и тд)
padding: ; - внутренние отступы (текст) со всех сторон (padding-top - работает так же, как и с margine)
    
text-decoration: ; - подчёркивание (none - нет, underline - снизу, overline - сверху, line-through -по середине)
text-decoration-color: ; - цвет подчёркивания
text-align: ; - расположение текста (left, right, center, justify)
text-transform: ; - изменение регистира (uppercase, lowercase, capitalize)
cursor: ; - изменение курсора при наведении (pointer - активная мышка)
content: ; - вывод текста для after и before
    attr() - для вывода атрибута

background-color: ; - цвет для заднего фона
background-image: url(""); - картинка для заднего фона
background-repeat: ; - настройка картинки
    no-repeat - картинка не повторяется
    repeat-x - повторяется по X 
    repeat-y - повторяется по Y
background-position ; - позиция картинки 
background: ; - сокращение предыдущих (background: red url("") no-repeat center)
background-size: ; - размер картинки
    cover - на 100%
background-attachment: fixed; - зафиксированная картинка

font-size: ; - размер шрифта (px, em)
font-style: italic; - текст наклоном 
font-weight: bold; - текст жирным (bolder - ещё жирнее)
font-family: ""; -  изменение шрифта 
word-spacing: ; - расстояние между словами
letter-spacing: ; - расстояние между символами 
line-height: ; - расстояние между строками
float: ; - позициирование блоков (left, right)
position - позиция 
    relative - сдвигает элемент относительно его обычного положения
    absolute - этот элемент можно будет поставить куда угодно, несмотря на другие блоки
    fixed - зафиксированный элемент, который будет оставаться при скроле колёсика мыши
z-index - слой, на котором будет находится объект


Flexbox - display: flex:

justify-content - выравнивает элементы горизонтально 
    flex-start - элементы выравниваются по левой стороне контейнера
    flex-end - элементы выравниваются по правой стороне контейнера
    center - элементы выравниваются по центру контейнера
    space-between - элементы отображаются с одинаковыми отступами между ними
    space-around - элементы отображаются с одинаковыми отступами вокруг них
align-items - выравнивает элементы вертикально
align-self - выравнивание к определенному элементу
    flex-start - элементы выравниваются по верхнему краю контейнера
    flex-end - элементы выравниваются по нижнему краю контейнера
    center - элементы выравниваются вертикально по центру контейнера
    baseline - элементы отображаются на базовой линии контейнера
    stretch - элементы растягиваются, чтобы заполнить контейнер
flex-direction - направление, в котором будут расположены элементы в контейнере
    row - элементы размещаются по направлению текста
    row-reverse - элементы отображаются в обратном порядке к направлению текста
    column - элементы располагаются сверху вниз
    column-reverse - элементы располагаются снизу вверх
order - указывает порядок flex-элемента
flex-wrap - указывает, нужно ли чтоб элементы принудительно находились в одном ряду или автоматически переносились
    nowrap - размеры элементов устанавливаются автоматически, чтобы они поместились в один ряд
    wrap - элементы автоматически переносятся на новую строку
    wrap-reverse - элементы автоматически переносятся на новую строку, но строки расположены в обратном порядке
flex-flow - комбинирование flex-direction и flex-wrap
align-content - выравнивает ряды flex-контейнера внутри него (работает только, если элементы расположены больше чем в один ряд)
    flex-start - ряды группируются в верхней части контейнера.
    flex-end - ряды группируются в нижней части контейнера.
    center - ряды группируются вертикально по центру контейнера.
    space-between - ряды отображаются с одинаковыми расстояниями между ними.
    space-around - ряды отображаются с одинаковыми расстояниями вокруг них.
    stretch - ряды растягиваются, чтобы заполнить контейнер равномерно.
text-align - выравнивает все строки 


Добавление стилей HTML (делается в HTML файле):
Пример: <p class="Лох"><p>

class="" - название повторяется 
id="" - название в id не должно больше повторяться


Псевдоклассы и псевдоэлемены:
Пример ."название элемента":hover

: - псевдокласс

hover - стиль при наведении на текст
active - стиль при нажатии на текст
visited - стиль к ссылкам, по которым мы уже переходили
focus - стиль при вводе текста
disabled \
enabled - стили с пометкой в input
required / 
empty - для text (пустые)
not(:empty) - для text (не пустые)


:: - псевдоэлемент

after - текст, который находится в конце и нельзя выделить
before - текст, который находится в начале и нельзя выделить
first-letter - обращение к первому символу абзаца
first-line - обращение к первой линии абзаца 
first-of-type - обращение только к самому первому тегу 
last-of-type - обращение только к самому последнему тегу (используется через одну :)
